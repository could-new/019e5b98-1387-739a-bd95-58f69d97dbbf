# SimpleChat

A very simple and lightweight chat application with a clean UI.

## Features
- **Phone number login (OTP):** Simple authentication flow with phone and OTP screens.
- **Basic profile:** Set up your name and profile picture.
- **User list:** View a list of your contacts to start chatting.
- **One-to-one chat:** Simple text messaging interface with real-time style presentation.
- **Clean UI:** Lightweight, no-nonsense design focusing on the essential features.

## Getting Started

1. Ensure you have Flutter installed.
2. Clone the repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the app on your preferred device or emulator.
